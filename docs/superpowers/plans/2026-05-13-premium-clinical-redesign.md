# NexGenCure Premium Clinical Redesign Plan

## Goal

전체 홈페이지를 단순한 정보형 화면에서 프리미엄 클리니컬 브랜드 사이트처럼 보이도록 개선한다. 기준은 맑은 화이트/블루 베이스에 샴페인 골드 포인트, 여백감 있는 고급 타이포그래피, 유리감 있는 헤더와 카드, 제품/기술/회사 소개가 더 신뢰감 있게 보이는 디자인 시스템이다.

## Scope

- 공통 헤더, 내비게이션, 버튼, 푸터 톤 개선
- 메인 슬라이드와 각 서브 페이지 히어로의 깊이감 강화
- 제품 카드, 빠른 메뉴, 프로세스, 인증/문의 카드에 프리미엄 패널 스타일 적용
- 모바일에서도 글씨와 버튼이 겹치지 않도록 반응형 보정

## Verification

- 디자인 시스템 토큰과 핵심 스타일을 확인하는 Node 테스트 추가
- 기존 콘텐츠/페이지 구조 테스트 유지
- 정적 검증 스크립트에 프리미엄 스타일 존재 여부 추가
- 로컬 서버에서 데스크톱/모바일 브라우저 스크린샷으로 시각 검수
